HTML TAGS

The structure and presentation of material on a webpages are specified by HTML tags.

  • <h1> to <h6> - Headings
  • <p> - Paragraphs
  • <a> - Anchor (Hyperlink)
  • <img> - Image
  • <ul> and <ol> - Lists
  • <li> - List Item
  • <div> - Division
  • <span> - Inline Division
  • <table> - Table
  • <form> - Form

CSS PROPERTIES

CSS properties style the appearance of HTML elements.

  • color
  • font-family
  • margin
  • padding
  • background-color
  • border
  • text-align
  • display
